## Installation

```
1. Clone the repository from ziqitza_vts-web.
        Repository Name:- ziqitza_vts-web
        Clone command:- git clone https://shivanisehgal@bitbucket.org/daffodilsoftware/ziqitza_vts-web.git

2. Checkout to the develop branch.
        Command:- git checkout develop

3. Take pull from the develop branch to get the latest code.
        Command:- git pull origin develop

4. Install all the dependencies.
        Command:- npm i
        (If any vulnerabilities found kindly run the following command:- npm audit fix)

5. Start the application.
        Command:- npm start

6. The application can now be accessed at http://localhost:8000.
```

## Build

```
- Development Build:-
        npm run build-development

- Staging Build:-
        npm run build-staging

- Production Builld:-
        npm run build-production

Note:- Install the dependencies correctly before running build command.
```
